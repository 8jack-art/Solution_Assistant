# 工程其它费用算法记录

## B2 建设管理费

### 1. 建设工程监理费

**基数定义：**
- 工程费用 = 建设工程费 + 安装工程费

**收费基价表（万元）：**

| 序号 | 工程费用 | 收费基价 |
|------|----------|----------|
| 1 | 500 | 13.20 |
| 2 | 1000 | 24.08 |
| 3 | 3000 | 62.48 |
| 4 | 5000 | 96.64 |
| 5 | 8000 | 144.80 |
| 6 | 10000 | 174.88 |
| 7 | 20000 | 314.72 |
| 8 | 40000 | 566.56 |
| 9 | 60000 | 793.12 |
| 10 | 80000 | 1004.64 |
| 11 | 100000 | 1205.60 |
| 12 | 200000 | 2170.00 |
| 13 | 400000 | 3906.08 |
| 14 | 600000 | 5468.48 |
| 15 | 800000 | 6926.72 |
| 16 | 1000000 | 8312.08 |

**计算规则：**
- 工程费用处于两个数值区间的，采用直线内插法计算
- 公式：`收费基价下限 + (收费基价上限 - 收费基价下限) × (工程费用 - 下限值) / (上限值 - 下限值)`

**算例：**
- 工程费用 = 15725.30 万元
- 计算：`174.88 + (314.72 - 174.88) × (15725.30 - 10000) / (20000 - 10000)`
- 结果：254.94 万元

### 2. 项目前期工作咨询费

**基数定义：**
- 估算总投资额（项目总资金）

| 咨询评估项目 | 500万以下 | 500万~1000万 | 1000万~3000万 | 3000万~1亿 |
|--------------|-----------|--------------|---------------|------------|

| 二、编制可行性研究报告 | 1.6~3.2 | 3.2~4.8 | 4.8~9.6 | 9.6~22.4 |
| 三、评估项目建议书 | 0.4~0.8 | 0.8~1.2 | 1.2~3.2 | 3.2~6.4 |
| 四、评估可行性研究报告 | 0.8~1.2 | 1.2~2.0 | 2.0~4.0 | 4~8 |
| 五、初步设计文件评估咨询 | 0.8~1.2 | 1.2~2.0 | 2.0~4.0 | 4~8 |

| 咨询评估项目 | 1亿~5亿 | 5亿~10亿 | 10亿~50亿 | 50亿以上 |
|--------------|---------|----------|-----------|----------|
| 一、编制项目建议书 | 11.2~29.6 | 29.6~44 | 44~80 | 80~100 |
| 二、编制可行性研究报告 | 22.4~60 | 60~88 | 88~160 | 160~200 |

**计算规则：**
- 根据估算总投资额在对应区间内用直线插入法计算

**算例：**
- 估算总投资额 = 40043.232 万元
- 编制项目建议书 = `11.2 + (29.6 - 11.2) / (50000 - 10000) × (40043.232 - 10000)`
- 结果：25.02 万元

## B5 勘察设计费

**基数定义：**
- 工程费用 = 建设工程费 + 安装工程费

### 勘察费

| 子项 | 费率 | 计算公式 |
|------|------|----------|
| 初勘及详细勘察费 | 0.3% | 工程费用 × 0.3% |
| 施工勘察费 | 1.2% | 工程费用 × 1.2% |
| **勘察费合计** | - | 两者之和 |

### 设计费

**基本设计费收费基价表（万元）：**

| 序号 | 工程费计费额 | 收费基价 |
|------|--------------|----------|
| 1 | 200 | 8.10 |
| 2 | 500 | 18.81 |
| 3 | 1,000 | 34.92 |
| 4 | 3,000 | 93.42 |
| 5 | 5,000 | 147.51 |
| 6 | 8,000 | 224.64 |
| 7 | 10,000 | 274.32 |
| 8 | 20,000 | 510.12 |
| 9 | 40,000 | 948.60 |
| 10 | 60,000 | 1,363.68 |
| 11 | 80,000 | 1,764.09 |
| 12 | 100,000 | 2,154.06 |
| 13 | 200,000 | 4,005.72 |
| 14 | 400,000 | 7,449.03 |
| 15 | 600,000 | 10,707.75 |
| 16 | 800,000 | 13,852.26 |
| 17 | 1,000,000 | 16,914.42 |
| 18 | 2,000,000 | 31,454.01 |

**系数设置：**
- 工程复杂程度调整系数：简单工程 0.85，一般工程 1.0，复杂工程 1.15，特别复杂工程 1.3
- 附加系数：1.10
- 复杂度系数默认值：1.0

**计算公式：**
- 基本取费 = 直线内插法计算
- 基本设计费 = 基本取费 × 复杂度系数 × 附加系数
- 竣工图编制费 = 基本设计费 × 8%
- 设计费 = 基本设计费 + 竣工图编制费

**算例（工程费用 = 15725.30 万元）：**
- 基本取费 = `274.32 + (510.12 - 274.32) / (20000 - 10000) × (15725.30 - 10000)` = 409.32 万元
- 基本设计费 = 409.32 × 1.0 × 1.10 = 450.25 万元
- 竣工图编制费 = 450.25 × 8% = 36.02 万元
- 设计费 = 450.25 + 36.02 = 486.27 万元
- 勘察费 = 235.88 万元
- **勘察设计费 = 235.88 + 486.27 = 722.15 万元**

**勘察设计费总计 = 勘察费 + 设计费**
